Как добавить категорию?

Если Вы собираетесь добавлять товары вручную, сначала создайте структуру категорий вашего каталога, чтобы затем помещать товары в нужную категорию.

{% if category_tree %}
Категории товаров
    {% for category, level in category_tree %} {# #}
  • {% nesting_level level %}{{ category.name }}
    • {#     #} {% endfor %}
{% else %}

Пример структуры каталога:

-Женская одежда
  • ---Верхняя одежда
  • -------Пальто
  • -------Пуховики
  • -------Шубы
  • ---Платья
  • -------Вечерние платья
  • -------Повседневные платья
{% endif %}
